# Application structure

Your application have no base structure, but this an example of application structure:

```
controllers/
    main.controller.js
forms/
    User.form.js
models/
    object.js
resources/
    css/
        index.css
    img/
        img.jpg
templates/
    index.twig
views/
    main.view.index.js
main.js
routes.js
```
